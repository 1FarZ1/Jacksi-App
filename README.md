## Jaksi-Task-App

- This is a simple task app i made for the Jaksi Interview, it is a simple task app that allows you to add,Get Products, filter them , and change the list , add images ... ect

## Screenshots

## Features

- Add Product
- Get Products
- Filter Products
- Change View Style
- Add Images
- Products Stored in Local Storage
